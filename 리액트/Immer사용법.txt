###### Immer ######


- 리액트에서 배열이나 객체를 업데이트 해야 할 때에는 직접 수정하면 안되고 불변성을 지켜주며 업데이트 하는 것이
중요하다.
- 불변성을 지켜주기 위한 코드를 작성하다 보면 코드가 복잡해져서 한 눈에 보기 어려워질 수가 있다. 이때
immer라는 라이브러리를 사용하면 된다
- immer 사용하면 우리가 상태를 업데이트 할 때, 불변성을 신경쓰지 않으면서 업데이트를 해줘도 immer가 불변성관리를
대신 해준다.

사용법: 프로젝트에 $ yarn add immer (npm install immer) 다운 -> import produce from 'immer';
코드의 상단에서 immer를 import해오기 보통 produce라는 이름으로 사용한다.
- produce 함수를 사용 할 때에는 첫번째 파라미터에는 수정하고 싶은 상태,
 두번째 파라미터에는 어떻게 업데이트하고 싶을지 정의하는 함수를 넣어준다.
 -- 두번째 파라미터에 넣는 함수에서는 불변성에 대해서 신경쓰지 않고 그냥 업데이트 
 해주면 다 알아서 해준다
