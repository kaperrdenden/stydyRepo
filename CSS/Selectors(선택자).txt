1. 전체 선택자 * 유니버셜 셀렉터
2. type셀렉터(엘리먼트,태그 셀렉터)
3. 클래스 셀렉터
4. id 셀렉터 한페이지당 하나만 사용가능하다
5. 속성 선택자
    --태그[속성] 해당 속성을 들고있는 모든 태그
    --태그[속성=값] 해당 속성의 값인 모든 태그
    --[attr^=value]	li[class^="box-"]
    --해당 속성이 해당 밸류로 시작하는 값들
    --[attr$=value]	li[class$="-box"]	Matches elements with an attr attribute whose value ends with value.
    --[attr*=value]	li[class*="box"] class내부에 box라는 글자가 어디든 존재하면
    --li[class^="a" i] 이렇게 ]닫기 전에 i를 써주면 아스키코드를 동등하게 간주헤준다
    --type tag(div)
